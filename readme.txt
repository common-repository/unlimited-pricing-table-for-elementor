Unlimited Pricing Table for Elementor. 
== Description ==
**Unlimited Pricing Table for Elementor** is a powerful Elementor addons plugin designed to help you create stunning pricing tables quickly and easily, without any programming skills. With a variety of layout options and customization features, you can showcase your products and services in the best possible light.

**Features:**
- **Responsive Design**: Ensure your pricing tables look great on all devices.
- **Easy to Use**: No coding required. Add features to your plans with a simple, user-friendly interface.
- **Multiple Layouts**: Choose from Table, Toggle, and Single layouts to suit your needs.
- **Custom Colors and Styles**: Fully customize the appearance of your pricing tables to match your brand.

**Available Layouts:**
1. **Table**: Display your pricing plans in a traditional table layout.
2. **Toggle**: Allow users to toggle between monthly and yearly pricing.
3. **Single**: Highlight a single pricing plan prominently.

**PRO Features**
Currently, there is no pro version available for this plugin.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/unlimited-pricing-table-for-elementor` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Edit a page with Elementor and search for pricing tables.
4. Add the "Pricing Table" widget to your page and customize it to your liking.

== Frequently Asked Questions ==

= How do I use Unlimited Pricing Table for Elementor? =
After installing and activating the plugin, open Elementor and search for the "Pricing Table" widget. Drag it to your page and start customizing your pricing table.

= The Elementor editor fails to load or is not working. What should I do? =
This issue is often related to your server's PHP settings. Increase your server PHP memory limit from the `wp-config.php` file or `php.ini` file. If you're unsure how to do this, contact your hosting provider and ask them to increase the following settings:
* PHP memory_limit = 512M
* max_execution_time = 300
